Projet de fin d'études

BannerStudio

BannerStudio (Mai-Juin 2021)

Outil de présentation de bannières Web animées. Réalisé pour l'agence DSPLY.

Spécifications techniques :

Outil interne permettant à l'agence de présenter les bannières pub aux clients.

BannerStudio facilite la livraison des bannières finies, de manière sécurisée et facilitée, tout en offrant une présentation sobre et élégante à l'utilisateur.

Appli web réalisée avec Symfony 5.2.9 / PHP 8 / jQuery / SASS / et Gulp 4.

Les bannières pub sont uploadées par les Admins sous forme de fichiers zip, lesquels sont ensuite associés à une campagne publicitaire, dézippés, associés à un format de bannière (pour adapter l'affichage notamment), puis affichés dynamiquement côté client.

L'application est développée desktop-first car la plupart des clients vont la consulter depuis un ordinateur, cependant, elle est responsive et s'adapte à un support de type smartphone ou tablette.

L'utilisateur a la possibilité de faire une récupération de mot de passe en cas d'oubli. Il a également accès à une page profil où il peut modifier son Nom / Prénom / mot de passe.

Compétences couvertes par l'application:

Modélisation de base de données / schéma fonctionnel de l'application et diagramme de classes.

Créer une base de données MySql.

Développer les composants d'accès aux données (CRUD) avec Symfony 5 (campagne / utilisateurs et leur statut / formats de bannières).

Développer la partie back-end d'une application Web en respectant les recommandations de sécurité et les recommandations/outils Symfony.

Développer le composant d'upload de fichiers.

Développer le rendu côté client / le téléchargement des fichiers par le client.

Rendre l'interface dynamique et responsive.
