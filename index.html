Exemples de réalisations

Les projets orientés Front-end

eCo2 Game Store

Concept : site Web de vente de jeux récents et anciens, dématérialisés, pour un jeu acheté, un arbre planté.

Specifications techniques :

Réalisation grâce au framework React JS sur la logique "single page application".

Le site comporte une partie admin fonctionnelle. Elle permet l'ajout, la modification et la suppression des infos d'un jeu et son illustration.

Site Web 100% responsive. Notamment grâce aux média queries.

Le Blog Auto

Blog automobile basé sur le CMS Wordpress.

Specifications techniques :

Réalisation d'une interface utilisateur avec une solution de gestion de contenu.

Développé en local (localhost) avec XAMPP.

Page d'accueil, page d'articles et page contact.

Mon premier Portfolio (fin 2020)

Création d'un portfolio / présentation d'articles web chargés dynamiquement grâce à la méthode AJAX.

Specifications techniques :

Les pages du site sont chargées dynamiquement avec AJAX.

Le contenu des articles est chargé dynamiquement, il est récupéré d'un fichier JSON.

Le site est séparé en 3 pages: Accueil / Présentation / Contact.

Le site est 100% responsive.

Les projets orientés Back-end

Passion Montres

Passion Montres est un site de vente de montres de prestige, réalisé avec le CMS Wordpress agrémenté de l'extension WooCommerce.

Specifications techniques :

Développé en local (localhost) avec XAMPP.

Site composé des pages: Accueil / Boutique / Actualités / Mon Compte / Politique de confidentialité.

Template Storefront légèrement personalisé.

Installation de l'extension WooCommerce pour intégrer un panier, la livraison, une facturation, des codes promos et toutes les fonctionnalités de paiement via PayPal.

Fiche produit composée d'une photo principale, deux photos complémentaires, les caractéristiques produit, la description et le prix TTC.

Actu wWweb

Actu wWweb est un blog d'actualité sur des sujets divers en rapport avec le Web. Le site est réalisé en PHP procédural, et comporte une administration complète qui permet de gérer les articles.

Specifications techniques :

Article composé de: titre / auteur / image d'illustration / contenu / catégorie.

Administration qui permet: l'ajout / la modification / la suppression d'un article. La barre de recherche est fonctionnelle.

L'accès de l'administration est soumis à authentification, login/mot de passe, et offre deux niveaux de grades : admin (plein pouvoirs) et user (il ne peut que consulter).

Compétences couvertes par l'application:

Créer une base de données MySql.

Développer les composants d'accès aux données (CRUD).

Développer la partie back-end d'une application Web en respectant les recommandations de sécurité.
